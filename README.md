## Overview

My finished Blog Preview Card project for Frontend Mentor. The project was built using HTML and styled with an external CSS stylesheet. It showcases a blog article preview card with a featured illustration, publication date, category tag, article title, description, and author profile section. The design uses a centered card layout with custom Figtree fonts, spacing, shadows, border styling, and hover effects to closely match the original Frontend Mentor design. This project helped me practice Flexbox, responsive layouts, spacing consistency, typography, custom font integration using `@font-face`, and overall frontend styling fundamentals.

## My process

I began by creating the overall card structure using semantic HTML elements for the image, text content, and profile section. Once the structure was in place, I focused on styling the card by adding the background color, border radius, shadows, padding, and spacing to closely match the original Frontend Mentor design. After the main layout was complete, I implemented Flexbox to properly center the card on the page and align the profile section elements. I then incorporated custom Figtree fonts using `@font-face`, which was my first experience working with local static fonts and different font weights. Finally, I refined the project by adjusting spacing consistency, typography, hover effects, responsive behavior, and other smaller visual details to create a cleaner and more polished final result.

### Built with

- Semantic HTML5 markup
- CSS custom properties
- Flexbox

### What I learned

This was my first time working with custom local fonts using `@font-face`. Because of that, I spent time researching and learning how custom fonts, font weights, and local font files are incorporated into a webpage. Through this process, I learned how to import static font files, assign different font weights, and apply them consistently throughout the project.
